# SystemIcon

SystemIcon component with different props.

#### Example

<Canvas of={SystemIcon.Default} />

## Usage

```ts
import {SystemIcon} from "@dreampipcom/oneiros";

const Example = () => {
  return (
      <SystemIcon
        icon={ESystemIcon.account}
        theme={"light"}
        variant={"medium"}
      />
  );
};

export default Example;
```

#### Arguments

- **icon** - A string that represents the name of the Icon.
- **theme** - A string that represents the color scheme of the Icon.
- **size** - A string that represents the size contents of the Icon.
