# Globals

Globals component with different props.

#### Example

<Canvas of={Globals.Light} />

## Usage

```ts
import { Globals } from "@dreampipcom/oneiros";

const App = () => {
  const [selectedTheme] = useContext({ selectedTheme: "lightTheme" })
  return (
      <Globals theme={selectedTheme} >
        <TheApp />
      </Globals>
  );
};

export default Example;
```

#### Arguments

- **children** A Theme Provider.
- **theme** - Color palette.
