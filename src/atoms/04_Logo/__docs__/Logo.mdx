# Logo

Logo component with different props.

#### Example

<Canvas of={Logo.Default} />

## Usage

```ts
import {Logo} from "@dreampipcom/oneiros";

const Example = () => {
  return (
      <Logo
        theme={"light"}
        variant={"medium"}
      />
  );
};

export default Example;
```

#### Arguments

- **theme** - A string that represents the color scheme of the Icon.
- **size** - A string that represents the size contents of the Icon.
