# TextInput

TextInput component with different props.

#### Example

<Canvas of={TextInput.Outlined} />

## Usage

```ts
import {TextInput} from "@dreampipcom/oneiros";

const Example = () => {
  return (
      <TextInput
        variant={ETextInputVariant.OUTLINED}
        label="Label"
        placeholder="This is a long placeholder."
        icon={ESystemIcon.account}
        onSelect={(e) => {}}
      />
  );
};

export default Example;
```

#### Arguments

- **variant** - A TextInput variant option (currently should default to OUTLINED only.
- **label** - A TextInput label string.
- **placeholder** - A TextInput placeholder/hint string.
- **icon** - A TextInput ESytemIcon name.
- **onChange** - A TextInput onChange callback.
